# AI-Driven 5G Network Digital Twin for Predictive Network Optimization

A simulation-based **Digital Twin system for cellular networks** that models network behavior in software and enables **predictive congestion analysis and AI-assisted optimization** without affecting real infrastructure.

---

## Key Features

- **Digital Twin Architecture** for modeling cellular network behavior
- **Telemetry Simulation** for dynamic metrics such as user density, latency, and traffic demand
- **Network State Model** to maintain a real-time representation of the system
- **What-If Simulation Engine** for forecasting network congestion scenarios
- **AI-Assisted Optimization** for generating load-balancing recommendations
- **FastAPI Service Layer** exposing REST endpoints for system interaction

---

## System Architecture

The platform follows a **modular service-oriented architecture** consisting of:

- Simulation Engine
- Network State Model
- AI Analysis Module
- Execution Engine
- FastAPI Backend Service

## API Endpoints

| Endpoint | Description |
|--------|-------------|
| `/state` | Retrieve current network state |
| `/simulate` | Run simulation for future network behavior |
| `/analyze` | Perform congestion risk analysis |
| `/execute` | Apply optimization actions |

## Purpose

This project demonstrates **system design, network simulation, and AI-assisted decision systems**, showcasing how digital twins can be used for **telecom network optimization and intelligent infrastructure management**.
